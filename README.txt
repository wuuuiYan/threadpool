1) 打开
不要在当前文件夹下鼠标右键选择使用Visual Studio 打开

正确的打开方式是：
	先从桌面打开软件，然后在软件的开始界面选择打开项目或解决方案
	然后选择当前目录下的threadpool.sln文件打开

2) 生成
在Visual Studio 2019的界面的解决方案资源管理器中，右键threadpool，选择 生成 或 重新生成
		运行程序：Ctrl + F5 或 调试 -> 开始执行(不调试)
		调试程序：F5 或 调试 -> 开始调试

3）提示
	使用解决方案资源管理器窗口添加/管理文件
	使用团队资源管理器窗口连接到源代码管理
	使用输出窗口查看生成输出和其他消息
